This file simulates the augmented Krugman model and generates
a test dataset for the empirical strategy. In order to start
the simulations, change the folder name in the main file and
set required parameters (number of countries, sectors, years).

Content:
1)Krugman-Augmented_ext.m - Main file, set parameters here
2)f_soleq_ext.m - Solves for equilibrium labor allocation
3)f_solw.m - Solves for equilibrium wages
4)f_output.m - Reorganizes equilibrium objects in output tables
5)Output folder - will contain output objects
